
#  Name Info Service

Сервис на Go для определения пола, возраста и национальности по имени, с сохранением результатов в базу данных.

## Описание
HTTP-сервер, который:
- Принимает имя через API
- Запрашивает данные у внешних API
- Сохраняет результат в PostgreSQL
- Возвращает JSON-ответ с собранной информацией

## Технологии
- Go
- `net/http`
- `github.com/jmoiron/sqlx`
